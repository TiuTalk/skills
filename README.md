# skills

My personal Claude Code skills, installable via [skills.sh](https://skills.sh).

## Install

```bash
npx skills add TiuTalk/skills
```

## Skills

### tdd

Structured TDD workflow that decomposes features into small specs and implements them one at a time through the strict red/green/refactor cycle.

### brainstorm-prd

Walks through an adaptive interview about a feature or large change, then writes a Product Requirements Document (PRD).

### project-manager

Discovers, clarifies, and structures requirements — from any source (developer ideas, client briefs, vague requests).
